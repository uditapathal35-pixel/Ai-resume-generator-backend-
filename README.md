# 🚀 AI Resume Generator Backend

## 📌 Features
- User Signup & Login
- Generate Resume
- Store Resume in Database
- Fetch All Resumes

## 🛠 Tech Stack
- Python
- Flask
- SQLite
- SQLAlchemy

## ▶️ How to Run

1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt

3. Run server:
   python app.py

## 📡 API Endpoints

### Signup
POST /signup

### Login
POST /login

### Generate Resume
POST /generate-resume

### Get All Resumes
GET /resumes

## 📷 Sample Output
Resume generated using user input.
